# 循环遍历 Map

用于遍历 Map 对象，并对其中的每一项进行操作。

![alt text](/img/data-operate/loop-map.png)

**Map 对象**

输入或选择一个待设置的`Map 对象`

**Map 键**

`Map 对象` 中的每一项的键值，即 `数据key`

**Map 值**

`Map 对象` 中的每一项的键值，即 `数据值`

### 使用示例

![alt text](/img/data-operate/loop-map-example.png)

此示例中，遍历 `Map 对象` 中的每一项，并将 `Map 键` 与 `Map 值` 打印到控制台。
