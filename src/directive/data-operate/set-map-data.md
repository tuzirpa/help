# 设置 Map 对象数据键值对

用户在 Map 对象中设置键值对，可以将键值对添加到 Map 对象中，也可以修改 Map 对象中已有键值对的值。

![alt text](/img/data-operate/set-map-data.png)

**Map 对象**

输入或选择一个待设置的 Map 对象

**数据 Key**

定义键值对的键名

**数据值**

定义键值对的值

### 使用示例

![alt text](/img/data-operate/set-map-data-example.png)

在上图中，我们将 Map 对象 `mapObj` 中的键值对：数据 key 设置为 `名字`， 数据值 `李四` 。
