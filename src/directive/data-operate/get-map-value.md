# 获取 Map 对象的键值

用于获取 Map 对象中指定的键值。

![alt text](/img/data-operate/get-map-value.png)

**Map 对象**

输入或选择一个待设置的`Map 对象`

**数据 Key**

输入`Map 对象`中待获取键值的键对应的`数据 Key`

**返回 Map 的 Value**

将获取到的键值对的值，保存到变量中
