# 指令名称：页面元素滚动
## 指令说明：元素$\{element\}滚动到底部，每次滚动$\{scrollDistance\}，延迟$\{scrollDelay\}毫秒
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | element | 页面元素对象 | variable |  |
 | scrollDistance | 每次滚动距离 | number |  |
 | scrollDelay | 滚动延迟 | number |  |


## 指令输出：

#### 没有输出参数
