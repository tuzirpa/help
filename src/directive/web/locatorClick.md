# 指令名称：CSS定位器点击元素
## 指令说明：在页面$\{page\}中，使用$\{selector\}定位到元素并点击
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | page | 网页对象 | variable |  |
 | selector | CSS或XPath选择器 | string | 请输入CSS或XPath选择器 \(例如: \#id, .class, input\[type="text"\], //div/span\) |
 | timeout | 超时时间 | number |  |
