# 指令名称：发送一次完整按键（按下+松开）
## 指令说明：发送键盘按键 $\{keyCode\} 按下 $\{time\} 毫秒后松开 
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | page | 网页对象 | variable |  |
 | keyCode | 键值 | string | 请输入需要发送的键值 |
 | time | 按下持续时间 | number | 请输入按键持续时间（单位：毫秒） |


## 指令输出：

#### 没有输出参数
