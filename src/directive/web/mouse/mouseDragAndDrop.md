# 指令名称：鼠标拖动并释放
## 指令说明：在页面$\{page\}中, 鼠标拖动并释放, 从\($\{startX\}, $\{startY\}\)到\($\{endX\}, $\{endY\}\), 延迟$\{delay\}毫秒
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | page | 标签页对象 | variable |  |
 | startX | 拖动起点X坐标 | number |  |
 | startY | 拖动起点Y坐标 | number |  |
 | endX | 拖动结束点X坐标 | number |  |
 | endY | 拖动结束点Y坐标 | number |  |
 | delay | 延迟 | number | 单位毫秒 |


## 指令输出：

#### 没有输出参数
