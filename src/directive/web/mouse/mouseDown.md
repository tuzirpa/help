# 指令名称：按下鼠标
## 指令说明：在页面$\{page\}中, 按下鼠标$\{button\}键
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | page | 标签页对象 | variable |  |
 | button | 选择鼠标按钮 | string |  |


## 指令输出：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | key | 按下鼠标的键 | web.mouseButton |  |
