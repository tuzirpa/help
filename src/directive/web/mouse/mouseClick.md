# 指令名称：鼠标点击
## 指令说明：在页面$\{page\}中，坐标\($\{x\}, $\{y\}\)处 单击$\{count\}次
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | page | 标签页对象 | variable | 请输入标签页对象 |
 | x | 鼠标的水平位置 | number | 请输入鼠标的水平位置 例如：100 |
 | y | 鼠标的垂直位置 | number | 请输入鼠标的垂直位置 例如：100 |
 | count | 点击次数 | number | 请输入单击次数 例如：1、2、3 |


## 指令输出：

#### 没有输出参数
