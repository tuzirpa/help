# 指令名称：上传文件
## 指令说明：将页面$\{browserPage\} 中选择元素$\{selector\}，并上传文件$\{filePath\}。
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | browserPage | 标签页对象 | variable |  |
 | selector | css选择器 | string | 请填写选择上传元素的css选择器 |
 | timeout | 超时时间 | number |  |
 | filePath | 文件路径 | string | 选择要上传文件路径 |
