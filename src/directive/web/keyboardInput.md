# 指令名称：键盘输入
## 指令说明：在页面$\{page\}中，选择元素$\{selector\}上按下键盘$\{keyboardKey\}，超时时间$\{timeout\}秒
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | page | 网页对象 | variable |  |
 | selector | css选择器 | string |  |
 | keyboardKey | 选择键盘按下的键 | string |  |
 | timeout | 超时时间 | number |  |


## 指令输出：

#### 没有输出参数
