# 指令名称：获取已打开标签页
## 指令说明：在页面$\{browserPage\}中获取匹配（根据$\{matchType\}）的标签页，保存到变量$\{page\}中
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | browserPage | 浏览器对象 | variable |  |
 | matchType | 匹配类型 | string |  |
 | url | 地址 | string |  |
 | title | 标题 | string |  |
 | index | 索引 | number |  |


## 指令输出：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | page | 标签页对象 | web.page |  |
