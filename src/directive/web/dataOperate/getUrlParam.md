# 指令名称：获取网址参数
## 指令说明：获取网址$\{url\}中的参数$\{paramKey\}，保存到变量$\{paramValue\}中,注：如有多个一样的参数只获取第一个
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | url | 网站 | string | 请输入网址 例如：https://www.baidu.com?word=hello |
 | paramKey | 参数名 | string | 请输入参数名 例如：word |


## 指令输出：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | paramValue | 参数值 | array |  |
