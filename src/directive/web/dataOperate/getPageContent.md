# 指令名称：获取标签页内容
## 指令说明：在页面$\{page\}中获取当前标签页的$\{webContent\}内容，保存到变量$\{content\}中
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | page | 标签页对象 | variable |  |
 | webContent | 获取内容 | string |  |


## 指令输出：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | content | 页面内容 | string |  |
