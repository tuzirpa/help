# 指令名称：页面中元素截图保存到本地
## 指令说明：在页面$\{browserPage\}中选中元素$\{selector\}，并对其进行截图，并保存到本地$\{fileSavePath\}。
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | browserPage | 标签页对象 | variable |  |
 | selector | css选择器 | string | 不填写则默认截取整个页面 |
 | fileSavePath | 文件保存路径 | string | 选择要文件保存路径 |


## 指令输出：

#### 没有输出参数
