# 指令名称：等待获取网页请求结果
## 指令说明：在网页$\{browserPage\}中获取监听请求结果，超时时间为$\{timeout\}秒，结果数据存入$\{responseData\}变量
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | browserPage | 网页对象 | variable |  |
 | listenerObj | 网页请求监听对象 | variable |  |


## 指令输出：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | responseData | 请求结果数据 | web.listenerObj |  |
