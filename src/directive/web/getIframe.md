# 指令名称：获取iframe
## 指令说明：在页面$\{browserPage\}中获取子页面url 包含 $\{iframeSelector\}保存到$\{webIframe\}
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | browserPage | 标签页对象 | variable | 选择标签页对象 |
 | iframeSelector | iframe的url | string | 请输入iframe的url（src属性） |


## 指令输出：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | webIframe | 对象 | web.iframe |  |
