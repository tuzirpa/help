## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | content | 写入内容 | variable |  |
 | fileName | 文件名 | string | 请填写文件名 |
 | dir | 文件存放目录 | string | 请选择文件存放目录 |
 | isCovered | 存在时文件是否覆盖 | boolean |  |
