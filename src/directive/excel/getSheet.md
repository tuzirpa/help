# 指令名称：通过名称获取一个工作表
## 指令说明：从$\{workbook\}对象读取一个工作表，并保存到$\{sheet\}。
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | workbook | excel对象 | variable |  |
 | sheetName | 工作表名称 | string | 工作表名称,默认值为Sheet1 |


## 指令输出：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | sheet | Excel工作表对象 | excel.sheet |  |
