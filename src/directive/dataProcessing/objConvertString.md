## 指令说明：将对象$\{obj\}转换为JSON字符串,并将结果赋值给$\{content\}
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | obj | 转换的对象 | variable | 请输入转换的对象 |


## 指令输出：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | content | 转换结果 | string |  |
