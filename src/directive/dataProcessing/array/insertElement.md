# 指令名称：插入数组元素
## 指令说明：数组对象$\{array\}中，在下标$\{index\}位置插入元素$\{value\}
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | array | 数组对象 | variable |  |
 | index | 数组下标 | number |  |
 | value | 数组元素值 | string | 请输入要插入的元素值 |
