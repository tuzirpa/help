# 指令名称：取数组项
## 指令说明：从数组$\{arrayObj\}中取出第$\{index\}项保存到$\{item\}
## 指令输入：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | arrayObj | 数组对象 | variable | 选择数组对象 |
 | index | 项下标（索引） | number |  |


## 指令输出：

 | 参数名 | 参数lebel | 参数类型 | 参数填写说明 | 
 | ------------- | ------------- | ------------- | ------------- |
 | item | 对象 | object |  |
